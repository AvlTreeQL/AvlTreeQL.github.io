bat——带语法高亮的cat天使

发表于 2018-12-10 分类于 Trick
让cat支持语法高亮

bat 介绍

bat 具有 cat 的所有功能，同时具有代码高亮，可以简单认为 bat 是 cat 的高级版本

bat 安装

  • linux apt 安装

sudo apt install bat

  • linux deb 安装

wget https://github.com/sharkdp/bat/releases/download/v0.9.0/bat-musl_0.9.0_amd64.deb
sudo dpkg -i bat-musl_0.9.0_amd64.deb 
# adapt version number and architecture

  • macOS 安装

brew install bat

  • 把cat替换成bat

alias cat='bat'

bat 个性化设置

显示可以使用的主题

bat --list-themes

如果要使主题成为默认主题，需要在shell启动文件中用export BAT_THEME ="Themes Name"来设定。
