获取网易云音乐外链

发表于 2018-12-12 分类于 Trick
记一次获取网易云音乐外链的步骤

获取音乐外链的作用

  • 音乐外链是可以直接用来下载音乐的，常见结尾是 http://xxxx.mp3
  • 音乐外链可于在线播放，常用于博客网页播放器中

如何获取音乐外链

  • 网上有很多第三方音乐外链获取网站，但是速度很慢
  • 使用各大云平台的对象存储服务
  • 这里我使用的网易云音乐的音乐外链，相对来说速度快很多

使用方法

首先打开下面的网易云音乐网站

https://music.163.com/

找到你想要的音乐，例如:

https://music.163.com/#/song?id=28815250

复制上面链接最后的id，放入下面链接的下划线中

http://music.163.com/song/media/outer/url?id=____.mp3
